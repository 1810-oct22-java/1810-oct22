Data Binding

Angular supports data binding - a mechanism for coordinating parts of a template (HTML) with parts of a component (TS). There are numerous types of Data Binding.

This is string interpolation, which uses curly braces to display data represented in the component or is used to evaluate expressions.

Routing

Routing is the mechanism by which Angular navigates between views, or pages, of your application. Much like a browser manages displaying different pages based on links clicked , the Angular Router is used to manage such navigations in your app. This is a crucial tool in building SPA's with angular.
