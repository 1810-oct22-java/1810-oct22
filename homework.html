Java

Java is an Object Oriented Programming Language, which means it deals with the create of objects, which is central to this language. The four main properties of Java are polymorhpism, inheritance, abstraction, and encapsulation. Polymorhpism allows an object to take on many forms. One example of polymorphism is method overloading. This is when a method with the same name is declared many times in the same class, the only difference between each method are either the number of parameters each contain, or the data types of the parameters. Another example of polymorhpism is method overriding. This is when you declare a method of the same name in a seperate class from where the method was first declared. It then the implementation of the method is changed. Finally there is covariant types. This is when an object is declared where the reference type is different from the instance type. Inheritance is when a class aqiuires state and behavior from another (parent) class. This is acheived by using the "extends" keyword and only one class can extend another. A class can also implement an interface, using the "implements" keyword. An interface is like a class, except they do not implement any methods. You can implement many interfaces. Abstraction is responsible for hiding complexity. This can be applied to either methods or classes. An abstract class cannot be instantianted and an abstract method has no implementation and any subclass must override them. Encapsulation is ability to hide data. Anything declared "public" is accessible from anywhere. Anything declared "protected" is only accessible from within the same package. Finally, anything declared "private" is only accessible from within the same class.
