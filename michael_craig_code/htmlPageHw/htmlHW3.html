Unions

There are two types of unions: union and union all. These unions are used to combine the results of two SELECT statements. A couple conditions must be met for the union operator to run successfully:
  1. Both SELECT statements must have the same number of expressions
  2. The corresponding expressions must have the same data type in the SELECT statements
